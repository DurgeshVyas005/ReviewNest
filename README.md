# ReviewNest

AI-powered guest review analysis platform for hospitality teams.

## Features

- **Sentiment Analysis** — Classify reviews as Positive, Neutral, or Negative
- **Theme Detection** — Identify themes: Food, Host, Location, Cleanliness, Value, Experience
- **AI Response Generation** — Auto-generate professional management responses
- **Review Analytics** — Dashboard with trends, charts, and KPI tracking

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- React Router DOM
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
